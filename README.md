# RunIt

## 프로젝트 배경

- 코로나 19 팬데믹 이후 건강을 되찾기 위한 런닝앱의 수요는 우상향 하고있습니다.
- 런닝앱에 그룹 별 리그 제도를 통해 꾸준한 런닝 습관을 가지고 경쟁하며 체력을 길러봅시다.

## 기능 소개

- 내가 뛴 경로를 확인하고 저장할 수 있습니대.
- 등록한 경로를 통해 과거의 나의 기록에 도전할 수 있습니다.
- 리그 제도를 통해 그룹원들과 경쟁하며 체력을 길러나갈 수 있습니다.

## 프로젝트 기간
- 2024.10.14~ 2024.11.29 (7주)
